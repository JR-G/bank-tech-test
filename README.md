Bank Tech Test
===

Instructions
---
### Project Setup
```
$ git clone https://github.com/JR-G/bank-tech-test
$ bundle
```

### Testing
```
$ rspec
$ rubocop
```

### Running the App
Launching it
```
$ irb
```
Creating a new account (it can also take an optional argument of an opening balance)
```
$ account = Bank.new / Bank.new(100)
```
To make a withdrawal
```
$ account.withdraw(100)
```
To make a deposit
```
$ account.deposit(100)
```
